# bun-webserver

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.0.29. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

---

INIT
bun init

RUN
bun run index.ts / bun index.ts
Bun supports js, jsx, ts, tsx
